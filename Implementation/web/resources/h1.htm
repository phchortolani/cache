Hilfe

Für wen sind die Hilfetexte dieser ersten Seite? Wer sieht sie?

Diese erste Seite wird nur von @net Manager Anwendern gesehen, die als System Administrator angelegt sind. Nach der Erstinstallation von @net sind die Anwender "Gast" und "User" automatisch die System Administratoren.

Spätere Anwender der mit @net erstellten Programme sehen diese Hilfetexte nicht. So bleibt die Hilfe für das Softwarehaus dem Kunden natürlich verborgen.
