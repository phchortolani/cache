Wie werden Nutzungsberechtigungen vergeben?

 Im Programm "Mandanten Verwaltung", "Erfassen Zugangsberechtigungen" werden die Nutzungsberechtigungen angelegt. Bedienerkennungen können aus Zahlen- oder Buchstabenkombinationen bestehen. @net Manager bietet eine hohe Flexibilität bei den Berechtigungen. So können in den Systemparametern (Parameter "BERECHTIGUNG") Gruppen von Bedienern angelegt werden. Im Menüpunkt "Erfassen Zugangsberechtigungen" kann jeder Bediener einer Gruppe zugeordnet werden. Möglich ist natürlich auch, für einen einzelnen Bediener ein neuen, separates Profil anzulegen, statt ihn einer Gruppe zuzuordnen. Näheres dazu ist in den Hilfetexten der "Mandanten Verwaltung" hinterlegt.
