Hilfe

 

Was ist Mandantenfähigkeit? Was hat Mandantenfähigkeit mit dem Internet zu tun?

 @net Manager ist mandantenfähig. Das bedeutet, ein und dieselbe Anwendung kann von verschiedenen Mandanten (z.B. verschiedene Firmen, die die Anwendung nutzen) anders gesehen und genutzt werden. Beispiel: Beim Start der Anwendung erscheint bei jedem Mandanten das eigene Firmenlogo. Das Softwarehaus kann so flexibel auf Kundenwünsche eingehen, ohne einen nennenswerten Mehraufwand zu haben.

Werden mit @net Manager Produkte erzeugt, die im Internet laufen, ergeben sich weitere Anforderungen an die Mandantenfähigkeit. Soll z.B. eine mit @net Manager erstellte Anwendung im Internet für beliebig viele Firmen bereitgestellt werden, ist die Datensicherheit eine wichtige Voraussetzung. Die Firmen, die die Anwendung über das Internet nutzen, sollen Zugang zu den Programmen, nicht aber zu den Daten anderer Nutzer/Firmen haben. Mit @net Manager könnte eine "Herzlich Willkommen bei ..." Seite gestaltet werden, die neue Kunden der Internet-Anwendung sehen, sobald sie die entsprechende Adresse aufgerufen haben. Nach der Eingabe einiger Daten (z.B. Firmenname, HRB-Nr. etc.) könnte die mit @net Manger erstellte Anwendung automatisch eine Mandantennummer für den neuen Kunden vergeben. Der Kunde erfaßt, von der Anwendung sicher geführt, selbst seine "Mandanten Layout Vorgaben", eröffnet damit einen "Account" und kann die Anwendung anschließend über das Internet nutzen. @net Manger bietet durch seine Internet-Mandantenfähigkeit viele Möglichkeiten der Software-Anwendung im Inter-/Intranet.
