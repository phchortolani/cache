Wie nutzt man Module?

@net Manager erlaubt den Einsatz von Modulen zum Aufbereiten verschiedener Programmstände. In jeder Anwendung können einzelne oder mehrere Klassen, Menüs, Formulare, Listen, einzelne Formularfelder etc. verschiedenen Modulen zugeordnet werden. So kann jedem Mandanten oder sogar jedem Benutzer ein Zugriff auf verschiedene Module erlaubt werden.
