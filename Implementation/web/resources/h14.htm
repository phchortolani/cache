Wie werden die erstellten Programmstände aus- und eingelesen?

 @net Manager erlaubt ein einfaches und schnelles Aus- und Einlesen von Programmständen. Im Menüpunkt "Mandantenverwaltung", "Programmstand zur Auslagerung aufbereiten" wird die Auslagerung angewählt. Dazu werden die aufzubereitenden Formulare ausgewählt. Es gibt die Möglichkeit, einzelne oder alle Formulare für einen Programmstand aufzubereiten. Sofern die Programmstrukur auf Module bezogen ist, macht es Sinn, ganze Module statt einzelner Formulare aufzubereiten. @net fragt auch, ob die Standardauslieferungsdateien (z.B. Sprachendatei o.a. Standarddateien gemäß Klassen Definition) mit ausgelagert werden sollen.

Zum Einlesen von Programmständen ist der Menüpunkt "Mandantenverwaltung", "Programmstand einlesen" anzuklicken. @net Manager überträgt dann den jeweils neuesten Programmstand aus der Zwischendatei in die aktuelle Prorammversion. Näheres zum Aus- und Einlesen von Programmständen ist in den entsprechenden Hilfetexten hinterlegt.
