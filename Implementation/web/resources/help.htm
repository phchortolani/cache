Hilfe zur Anmeldung
Zum erstmaligen Anmelden im @net Manager ist kein Passwort erforderlich. Sie haben die Wahl zwischen folgenden Anwendernamen:

"GAST" und "USER": Nach der Eingabe "GAST" und dem Klicken des Start-Buttons startet @net Manager die deutsche, nach der Eingabe von "USER" die englische Anwendung.

"TEST" und "DEMO": @net Manager kann auch mit den Anwendernamen "TEST" (deutsch) und "DEMO" (englisch) gestartet werden. Diesen Anwendern fehlt in den Programmen die Berechtigung zum Ändern der Daten (ideal für Demo-Versionen der mit @net Manager erstellten Anwendungen).

"UNKNOWN": Wird @net Manager ohne die Eingabe eines Namens gestartet, geht @net Manager vom "UNKNOWN" User aus. Dieser Benutzer ist standardmäßig angelegt und für Anwendungen vorgesehen, die keine Anmeldung mit Berechtigungen/Passwörtern vorsehen. Um die Sicherheit Ihrer Anwendung zu gewährleisten, muß der Bediener "UNKNOWN" aus der Bedienerliste gelöscht werden.


Hilfe zur Startseite
Die hier vorgestellte Startmaske (index.htm) von @net Manager soll lediglich als Beispiel verstanden werden. Jedes Softwarehaus kann eine eigene Startmaske designen. Wichtig dabei ist, daß die in den Eingabefeldern gesetzten Variablen ("YBED" für den Bediener und "YPWD" für das Passwort) übernommen werden.
