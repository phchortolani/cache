Wie erstellt man eine Anwendung?

Grundsätzlich erlaubt @net Manager die Erstellung von manuellen oder automatisch generierten Formularen, Listen, Untermasken etc. Auch Kombinationen aus z.T. manuell und z.T. automatisch erstellten Formularen, Listen usw. sind möglich. Sie alle können mit Buttons, Seitenreitern, Optionen usw. versehen werden (siehe Menüpunkt "Erstellen von Formularen und Listen" inkl. der dort hinterlegten Hilfetexte)..

Wenn @net Manager beim Erstellen von Formularen, Listen etc. automatisch auf Klassen Definitionen zugreifen soll, sind diese vor oder nach der Formular-/Listenerstellung anzulegen (siehe Menüpunkt "Klassen Definitionen").

Um die erstellten Formulare, Listen etc. in das Menü einzubinden und dadurch für den Anwender nutzbar zu machen, ist das Benutzer Menü zu erstellen. Nähere Informationen hierzu finden sich im Menüpunkt "Erstellen Programm Menü".
