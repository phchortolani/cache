Wie werden Klassen Definitionen erfaßt?

Die Klassen Definitionen werden im Programm "Erfassen Klassen Definitionen" erfaßt. Hier wird vordefiniert, wie die Datei später aussehen soll. Sind die Klassen Definitionen inkl. Primärschlüssel und Datenfelder erfaßt, kann @net Manager Vorschläge für Formulare, Listen etc. unterbreiten, die dann weiterbearbeitet werden können. Jede Klassen Definition braucht mindestens einen Namen und kann einen oder mehrere mindestens Primärschlüssel besitzen. Der Primärschlüssel kann angelegt werden um mehrdimensionale Globals zu erzeugen. Wenn kein Primärschlüssel angelgt wird wird die Datei automatisch Sequenziell. Um dann einen schnellen Zugriff auf die Datensätze zu erhalten sollte auf jeden Fall ein oder mehrere Datenfelder zu einen eindeutigen Key führen (Auftragsnummmer, Artikelnummer o.ä.).
