help for user-identification
You can use the username "USER" (english user) without password or user "GAST" (German user) as default user.
Username "TEST" or "DEMO" without password have no data edit access.

If no user is specified, the "UNKNOWN" user is assumed.
This user is reserved for applications that requires no user identification.
To ensure security, you have to delete the user "UNKNOWN" from the user list.
