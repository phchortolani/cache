Consulta de estoque por produto [ Voltar ]

Para consultar a quantidade existente em estoque de um determinado produto, clique em "Consulta de estoque por produto" dentro do menu "Estoque".

O sistema vai abrir a seguinte tela:

Dentro desta tela você pode conferir uma lista contendo todos os produtos cadastrados no sistema.

1º Passo. Se você já souber o código do produto, insira-o no campo "Produto". Se você não souber o código do produto, ao clicar no ícone , a seguinte janela pop-up será mostrada:


2º Passo. Procure o produto desejado e clique nele. A janela pop-up será fechada e o código do produto será preenchido automaticamente no campo "Produto":

3º Passo. Clique no botão para continuar. O resultado de sua pesquisa será mostrado:


Esta tela contém as especificações do produto, incluindo seu endereço no armazém, custo, conta contábil, quantidade, etc.

Se você quiser realizar outra pesquisa, poderá voltar para a página anterior clicando no botão .
