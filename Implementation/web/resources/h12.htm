Wie können die Layout Vorgaben angelegt und verändert werden?

 Nach der Installation von @net Manager ist automatisch der Mandant "0" angelegt, der für das System reserviert ist. Beim Erstellen einer Anwendung können im Menüpunkt "Mandantenverwaltung", "Mandanten Layout und Programmvorgaben" beliebig viele weitere Mandanten angelegt werden, deren Namen/Nummern bis zu 30 Zeichen lang sein können. Für jeden Mandanten werden dann die verschiedenen Optionen angelegt. So könnte jeder Mandant z.B. sein eigenes Firmenlogo beim Starten seiner Anwendeng sehen. Auch die Hintergrundfarbe, der Wasserzeicheneffekt, Hintergrundbilder, Textausrichtung, Schriftfarben und -größen, Pflichtfelder, Buttons usw. können von Mandant zu Mandant variieren.
