Conta corrente [ Voltar ]

Utiliza o formulário "Conta corrente" para vizualizar o histórico de todos os eventos referentes a um produto em um determinado período. O histórico inclui operações como recebimentos, requisições, realocações e saídas, assim como as quantidades, hora, data, prazo de validade, entre outras informações do produto. 

O fomulário "Conta corrente" encontra-se dentro do menu "Estoque" 


Ao clicar em "Conta corrente", a seguinte tela será exibida:



Siga os seguintes passos para o preechimento deste formulário:

1º Passo - Informe o produto para o qual deseja visualizar o histórico. Clique no botão "Procurar"  ao lado do campo "Produto" para visualizar a lista de todos os produtos existentes. Clique duas vezes no produto desejado para selecioná-lo.



2º Passo - Informar a data inicial . Clique no botão "Data"   ao lado do campo "Data inicial" para selecionar, com um clique, a data a partir da qual todos os eventos relacionados ao produto escolhido devem ser listadas.

3º Passo -  Informar a data final. Clique no botão "Data"   ao lado do campo "Data final" para selecionar, com um clique, a data final para o histórico.

4° Passo - Clique no botão "OK" 

Ir para o topo da página
