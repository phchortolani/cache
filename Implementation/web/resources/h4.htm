Hilfe

Wie hinterlegt man Hilfetexte für die Anwender?

Nachdem ein Formular, eine Liste etc. generiert worden ist, wird die Hilfe von @net Manager automatisch bereitgestellt. Das bedeutet, @net Manager setzt die Feldbezeichnungen des Formulars als Überschrift des Hilfetextes ein. Sofern der Ersteller des Formulars die Berechtigung zur Erfassung von Hilfetexten hat, braucht er nach oder während der Formularerstellung lediglich auf die entsprechende Überschriften zu klicken und kann dann die Texte formulieren. Dabei können .gif, .jpg und .avi Dateien eingesetzt werden, um die Hilfetexte mit Bildern, Sound und Videos zu untermalen.

Im Hilfetext kann wie folgt ein Link in ein anderes Formular eingebaut werden: Freistehende Zeile: HYPERLINK: Danach folgt der Link-Text, dann der Formularname. Optional kann danach ein fester KEY vorgegeben werden, danach der Text des Tool-Tips. Beispiel: HYPERLINK:Artikelstamm:INART::Zum Artikelstamm Soll sofort Artikel 1 angezeigt werden: HYPERLINK:Artikelstamm:INART:1:Zum Artikelstamm Ein Rücksprung ist jedoch leider nicht möglich, da der Key nicht richtig verwaltet werden kann.
