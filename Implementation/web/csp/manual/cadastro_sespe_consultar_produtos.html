Consultar Produtos [ Voltar ]

Este formulário permite a localização e conferência de dados dos produtos cadastrados.

O formulário "Consultar Produtos" encontra-se dentro do menu "Cadastros".


 

Ao clicar no formulário, o sistema exibirá a seguinte tela:


 

Siga os passos abaixo para pesquisar um produto:
 
1° Passo: no topo da tela, selecione na seção "Favoritos" o tipo de pesquisa desejada. As opções de pesquisa são:
 
  • Padrão: para visualizar todos os produtos cadastrados. Filtros de pesquisa:  
    • Cód. Produto; e 
    • Descrição Resumida;
  • Medicamentos: para exibir apenas os medicamentos cadastrados no sistema. Filtros de pesquisa: 
    • Cód. Produto; 
    • Descrição Resumida (inicia com) - Digite as iniciais da descrição do produto; 
    • Descrição Resumida (contém) - digite qualquer palavra-chave da descrição do produto; 
    • Grupo/Subgrupo; e 
    • Status
  • Materiais médico-hospitalares: para exibir somente os materiais médico-hospitalares cadastrados. Os filtros de pesquisa são os mesmos existentes em "Medicamentos".
  • Todos os Produtos: para uma visão detalhada de todos os produtos cadastrados no sistema. Os filtros de pesquisa são os mesmos existentes em "Medicamentos". 
 
   2° Passo: você pode utilizar os campos da seleção de campos como filtros para facilitar sua pesquisa. para mais informações sobre as funcionalidades da pesquisa, favor ver o manual Introdução ao Sistema.
 

 

Após utilizar os filtros, os resultados da pesquisa são exibidos automaticamente na tabela de resultados abaixo. Selecione com um clique o produto que deseja. Assim que selecionado, o produto em questão será aberto na tela "Cadastro de Produtos".
