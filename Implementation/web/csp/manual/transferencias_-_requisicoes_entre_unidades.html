Requisição entre Unidades [ Voltar ]

Esta tela tem por função permitir a realização de pedidos de transferência entre duas unidades da rede de saúde. 

O formulário "Requisição entre Unidades" encontra-se dentro do menu "Transferência".


  

Ao clicar no formulário, o sistema exibirá a seguinte tela:

   

  

Para criar uma requisição, siga os passos abaixo:

1º Passo: selecione o local de origem. Para abrir uma janela contendo uma lista dos possíveis locais de origem, clique no botão [Procurar]. Será mostrada a seguinte janela:

   

  

Selecione na lista o local principal, de onde o estoque será adquirido. Ao clicar no local desejado, a janela será fechada automáticamente e o sistema irá preencher o número de tal local no campo adequado:

   

  

2º Passo: determine o grau de prioridade. Selecione a prioridade de acordo com as características do pedido: Emergencial, Periódico e Eventual.

   

  

3º Passo: selecione a data de entrega. Neste campo, o usuário pode registrar a data desejada para a entrega. 

   

 

Existem várias maneiras de preencher este campo. Uma é simplesmente escrevendo a data no formato DD/MM/AAAA (dia, mês e ano). Outra maneira é clicando no botão [Data] para selecionar em um calendário a data desejada. Uma terceira forma é seguindo uma das seguintes instruções:

  • Data atual: digite o sinal . (ponto) e pressione a tecla "Enter" para que o sistema retorne a data atual;
  • Data do mês corrente: digite o dia do mês e pressione a tecla "Enter" para que o sistema retorne o mês e ano correntes;
  • Dias a contar da data atual: digite o sinal + (mais) ou - (menos) antes do número de dias em referência à data atual e pressione a tecla "Enter" para a data anterior ou posterior à data atual. Por exemplo: se a data atual é 20/12/2011, digite "+10" e pressione a tecla "Enter" para que o sistema retorne o dia 30/12/2011.

 

4° Passo: clique no botão  para salvar os dados inseridos. Assim que o registro é salvo, uma grade para a inclusão dos produtos do pedido de transferência é exibida na parte inferior da tela.

   

 

5º Passo: especifique os produtos e respectivas quantidades para o pedido. Clique no botão [Procurar] presente no campo "Produto" para selecioná-lo a partir de uma listagem de produtos cadastrados. No campo "Quant. requisitada", informe a quantidade desejada do produto selecionado. Pressione a tecla "Enter" ou "Tab" para ir de um campo a outro na grade e também para criar uma nova linha. 

Os demais campos da grade exibirão, futuramente, dados processuais do pedido, como a quantidade aprovada, unidade, quantidade pendente, etc.

6º Passo: clique no botão  [Salvar] para salvar os produtos e quantidades especificadas.

7º Passo: se todos os dados estiverem corretos, clique no botão  para realizar o pedido. Observe que, após a confirmação, o status do pedido passará de    para   .

 

Se desejar, clique no botão  para adicionar observações adicionais sobre esta requisição.
