Relatório de Produtos sem Movimentação [ Voltar ]

Este formulário permite que o usuário tenha uma visão geral dos produtos sem movimentação. Ao gerar o relatório, serão exibidos os produtos, de toda a rede ou de um local específico, que não tenham sido movimentados (isto é, recebidos ou expedidos) há um determinado número de dias.

O formulário "Relatório de produtos sem movimentação" encontra-se dentro do menu "Estoque".



O sistema exibirá a seguinte tela:


Para gerar uma lista contendo todos os produtos sem movimentação, siga os passos abaixo:


1º Passo: selecione um local para esta consulta. Se este campo for deixado em branco, o relatório incluirá todos os locais da rede;

2º Passo: escolha o programa do produto. Este campo é de preenchimento voluntário, se for deixado em branco mostrará os produtos de todos os programas (Nutrição, Tabagismo, Diabetes, etc.). Clique o botão [Procurar] para selecionar o grupo a partir de uma listagem de grupos cadastrados;

3º Passo: selecione, se for o caso, o grupo do produto. Se o campo for deixado em branco, o relatório incluirá todos os produtos do sistema, indepentemente do grupo;

4° Passo: informe o número de dias que o produto deve estar sem movimentação para constar no relatório. Este é o único campo de preenchimento obrigatório;

5º Passo: determine se a pesquisa deve incluir produtos zerados. Se desejar, marque a opção "Excluir itens zerados", o qual fará que o relatório ignore os produtos sem estoque;

6º Passo: se desejar, selecione que as movimentações internas sejam ignoradas. Ao clicar na caixa de seleção "Excluir Movimentação Interna", o relatório irá ignorar todas as movimentações de estoque internas;

7° Passo: escolha o tipo de ordenação do relatório. Os tipos de ordenação na qual o relatório pode ser gerado são (a) pelos códigos dos produtos, (b) pelas descrições dos produtos ou (c) pelos dias sem movimentação. Se este campo for deixado em branco, o relatório será ordenado pelos códigos dos produtos.

8º Passo: clique no botão . Os resultados de sua pesquisa serão exibidos na tela:


Exemplo de relatório de produtos sem movimentação há 90 dias

4º Passo: clique no botão  para ter acesso à tela do relatório "Produtos sem movimentação".

5° Passo: se desejar imprimir o relatório, clique no botão  [Imprimir] do navegador.
