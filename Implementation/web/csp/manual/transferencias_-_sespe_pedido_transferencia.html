Pedido de Transferência [ Voltar ]

 
Para registrar um pedido de transferência, clique em "Pedido de Transferência", que encontra-se dentro do menu "Transferência".


 
 

Ao clicar no formulário, o sistema exibirá a seguinte tela:


 

Siga os passos abaixo para efetuar um pedido de transferência:

1º Passo: informe os dados principais do pedido. Os campos em amarelo são obrigatórios. 

  • Pedido. Este campo apresenta um sinal de mais "+" por padrão. Desta forma, assim que o registro é salvo, é-lhe atribuído o próximo número de pedido disponível.
  • Status. É exibida neste campo a situação atual do pedido: Não processado, Aberto, Aguardando aprovação, Aguardando atendimento, Atendido e Encerrado.
  • Local Requisitante. Informe aqui o local para o qual o pedido está sendo realizado. O local onde o usuário se encontra é carregado automaticamente.
  • Programa. Se for o caso, selecione neste campo o programa da Secretaria de Saúde que quiser associar a este pedido de transferência (programas como, por exemplo, Saúde da Criança, Tabagismo, AIDS/DST, etc.).
  • Observações. Se desejar, deposite neste campo informações adicionais sobre o pedido de transferência.
  • Prioridade. Selecione a prioridade de acordo com as características do pedido: Emergencial, Periódico e Eventual.
  • Data para entrega. Neste campo, o usuário pode registrar a data desejada para a entrega. Dicas para preenchimento dos campos de data:
     
    • Data atual: digite o sinal . (ponto) e pressione a tecla "Enter" para que o sistema retorne a data atual;
    • Data do mês corrente: digite o dia do mês e pressione a tecla "Enter" para que o sistema retorne o mês e ano correntes;
    • Dias a contar da data atual: digite o sinal + (mais) ou - (menos) antes do número de dias em referência à data atual e pressione a tecla "Enter" para a data anterior ou posterior à data atual. 
     
    Exemplo: se a data atual é 20/10/2012, digite "+10" e pressione a tecla "Enter" para que o sistema retorne o dia 30/10/2012 
 
2° Passo: clique no botão  para salvar os dados inseridos. Assim que o registro é salvo, uma grade para a inclusão dos produtos do pedido de transferência é exibida na parte inferior da tela.
 

  

3º Passo: especifique os produtos e respectivas quantidades para o pedido. Clique no botão [Procurar] presente no campo "Produto" para selecioná-lo a partir de uma listagem de produtos cadastrados. No campo "Quant. pedida", informe a quantidade desejada do produto selecionado. Pressione a tecla "Enter" ou "Tab" para ir de um campo a outro na grade e também para criar uma nova linha - ou clique no botão  [Novo] acima da grade para criar uma nova linha.

Os demais campos da grade exibirão, futuramente, dados processuais do pedido, como a quantidade aprovada, local abastecedor, número da requisição gerada, etc.

4º Passo: clique no botão  para salvar os produtos e quantidades especificadas.

5º Passo: se todos os dados estiverem corretos, clique no botão  para realizar o pedido. Observe que, após a confirmação, o status do pedido passará de     para   .

Para visualizar o relatório do Pedido de Transferência, clique no botão . 

Como cancelar de um pedido de transferência 

 
Para efetuar o cancelamento integral de um pedido de transferência já confirmado, clique no botão que se encontra na barra de botões (ver imagem abaixo). Em seguida, informe o motivo do cancelamento na janela de cancelamento que se abrirá e confirme com o botão "OK".
 
 
 
 
Se desejar cancelar apenas um ou mais produtos do pedido, selecione a opção "Sim" no campo "Cancelar" referente à linha do(s) produto(s) do cancelamento e pressione o botão logo acima da grade (ver imagem abaixo). Em seguida, informe o motivo do cancelamento na janela de cancelamento que se abrirá e confirme com o botão "OK".
