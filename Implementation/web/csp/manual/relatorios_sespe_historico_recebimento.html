Histórico de Recebimentos  [ Voltar ]

Utilize esta tela para gerar um relatório contendo o histórico dos recebimentos já registrados no sistema. O relatório pode ser gerado para apenas um período de tempo, local, fornecedor, produto, entre outras opções. 
 
O formulário "Histórico de Recebimentos" encontra-se dentro do menu "Relatórios Gerenciais".
 

 


Ao clicar no menu, a seguinte tela será exibida:


Siga os passos abaixo para efetuar a consulta de histórico de recebimentos:
 
1º Passo: configure a pesquisa para obter os dados desejados. Os campos em amarelo são obrigatórios.

  • Data Inicial / Data Final. Clique no botão  para selecionar a data desejada. Dicas para preenchimento de campos de data:
    • Data atual: digite o sinal . (ponto) e pressione a tecla "Enter" para que o sistema retorne a data atual;
    • Data do mês corrente: digite o dia do mês e pressione a tecla "Enter" para que o sistema retorne o mês e ano correntes;
    • Dias a contar da data atual: digite o sinal + (mais) ou - (menos) antes do número de dias em referência à data atual e pressione a tecla "Enter" para a data anterior ou posterior à data atual. 
  • Local. Para recebimentos de um local específico, selecione aqui o local;
  • Tipo. Para um tipo determinado de recebimento, informe-o aqui;
  • Programa. Informe aqui um programa para que a consulta retorne apenas recebimentos associados a um determinado programa. Clique no botão  [Procurar] para selecionar o programa a partir de uma lista de programas cadastrados;
  • Produto. Você pode restringir a consulta para recebimentos de um produto específico. Informe-o neste campo. Clique no botão  [Procurar] para selecionar o programa a partir de uma lista de programas cadastrados;
  • Grupo. Você pode restringir a consulta para recebimentos de um grupo específico. Informe-o neste campo. Clique no botão  [Procurar] para selecionar o programa a partir de uma lista de grupos cadastrados;
  • Fornecedor. Você pode restringir a consulta para recebimentos de um fornecedor específico. Informe-o neste campo. Clique no botão  [Procurar] para selecionar o programa a partir de uma lista de fornecedores cadastrados;
  • N°. OF. Se desejar retornar recebimentos associados a uma Ordem de Fornecimento específica, informe aqui o número da mesma. Clique no botão  [Procurar] para selecioná-la a partir de uma lista de ordens de fornecimento existentes;
  • Formato: determine neste campo o tipo de relatório que deseja gerar: Completo ou Simplificado.


2° Passo: clique no botão para processar a consulta. Assim que é processada, os dados são exibidos na parte inferior da tela.



Para gerar o relatório, clique no botão .
 

 
 
 
Se desejar imprimir o relatório, clique no botão  [Imprimir] do navegador. Para voltar à tela anterior, clique em  [Voltar].
