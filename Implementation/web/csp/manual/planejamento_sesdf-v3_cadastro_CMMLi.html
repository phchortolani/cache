Cadastro de CMMLi [ Voltar ]

Esta tela permite o usuário cadastrar informações do Consumo Médio Mensal.

O formulário "Cadastro de CMMLi" encontra-se dentro do menu "Programação - Planejamento".
 
 
 
 
Ao clicar no formulário, o sistema exibirá a seguinte tela:
 
     
 
 
Esta tela possui algumas informações importantes. São elas:
  • O Total de Produtos Autorizados ao Local. Informa a quantidade de produtos por grupo que estão disponíveis para o local atual do usuário.
  • Total com CMMLi Cadastrado. Informa a quantidade de produtos no grupo que já possui um Consumo Médio Mensal informado.
  • Total sem CMMLi Cadastrado. Informa a quantidade de produtos no grupo que ainda não possui um Consumo Médio Mensal informado cadastrado.
1º Passo: configure os filtros da tela para localizar o grupo dos produtos desejado ou selecione com um clique por meio da listagem que aparece na tela.
  • Grupo do Produto. Informe aqui o código do grupo do produto. Clique no botão [Pesquisar] para selecioná-lo a partir de uma listagem contendo todos os grupos de produtos cadastrados.
  • Produto. Informe aqui o código do produto. Clique no botão [Pesquisar] para selecioná-lo a partir de uma listagem contendo todos os produtos cadastrados.
  • Ordenar Por:. Selecione aqui entre as opções: "Código", "Descrição" e "Grupo".
Observação: Ao ser adicionado no campo Grupo do Produto uma informação válida, você ao clicar no botão será direcionado para a tela de cadastro de CMMLi.

 
 
 
Ao selecionar o grupo você será redirecionado para a tela abaixo.

 
 
 
Nesta tela o usuário efetiva o cadastramento do novo CMMLi, nela são apresentadas algumas informações importantes sobre os produtos do grupo selecionado, como o Código, Descrição, Unidade, Grupo, Estoque, CMML e o CMMLi atual.
2º Passo: informe o novo CMMLi para cada produto desejado. No campo ilustrado abaixo.
 
 
 
Após informar no campo o novo Consumo Médio Mensal informado, clique no botão [Processar] para salvar as novas informações adicionadas.
