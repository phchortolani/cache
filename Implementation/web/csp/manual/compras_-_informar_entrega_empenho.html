Informar Entrega do Empenho [ Voltar ]

Utilize esta tela para cadastrar no sistema um empenho associado a uma ordem de fornecimento. 

O formulário "Informar entrega do empenho" encontra-se dentro do menu "Compras".


 

Ao clicar no formulário, o sistema exibirá a seguinte tela:


  

Para registrar um empenho, siga os passos abaixo:

1º Passo: localize e selecione com um clique a ordem de fornecimento desejada. Utilize os filtros disponíveis para auxiliar a localização da ordem. Os filtros disponíveis são: 

  • Número da OF. Informe aqui os caracteres contidos no código da ordem de fornecimento. Assim, a pesquisa retornará apenas registros que contêm o conteúdo digitado.
  • Fornecedor. Informe aqui o fornecedor desejado. Digite os caracteres iniciais do código do fornecedor para retornar os fornecedores cujos códigos se iniciem com o conteúdo digitado. Se desejar, clique no botão  [Procurar] para selecioná-lo a partir de uma listagem contendo todos os fornecedores cadastrados.

Após selecionar a ordem, a seguinte tela será aberta:

 

2° Passo: informe os detalhes do empenho. Os campos em amarelo são obrigatórios.

3° Passo: clique no botão  para salvar os dados registrados.
