Consulta de Estoque por Endereço [ Voltar ]

Esta tela permite o usuário consultar a quantidade de produtos existentes em estoque de um determinado endereço, intervalos de endereços e por um conjunto de endereços agrupados.
Para acessá-la clique no link "Consulta de Estoque por Endereço" dentro do menu "Estoque - Controle de Estoque".

 
 
 
O sistema abrirá a seguinte tela:

   
   
     
Para verificar a situação dos produtos baseado em seus endereços de armazenagem, siga os passos a seguir:
1º Passo: selecione o tipo de consulta que será realizada. O campo Tipos de consulta é de suma importância pois a partir dele os outros campos são liberados ou não. As opções deste campo são: "Consultar um endereço específico", "Consultar um intervalo de endereços" e "Consultar um grupo de endereços pelo seu prefixo".
  • Endereço. Este campo estará disponível apenas se no campo Tipos de consulta a opção selecionada for "Consultar um endereço específico". Entre neste campo com um endereço específico para realizar a consulta. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os endereços cadastrados.
  • Endereço inicia com. Este campo estará disponível apenas se no campo Tipos de consulta a opção selecionada for "Consultar um grupo de endereços pelo seu prefixo". Entre neste campo com a inicial dos endereços desejados para realizar a consulta. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os endereços cadastrados.
  • Desde o endereço...Até. Este campo estará disponível apenas se no campo Tipos de consulta a opção selecionada for "Consultar um intervalo de endereços". Entre nestes campos com um intervalo de endereços para realizar a consulta.

2º Passo: após preencher os filtros desejados clique no botão  para gerar a consulta. Os resultados irão aparecer na tela conforme imagem abaixo.

 
 
 
3º Passo (Opcional): clique no botão [Relatório]. Este botão abrirá o relatório "ESTOQUE POR ENDEREÇO", que contém os resultados da pesquisa.
