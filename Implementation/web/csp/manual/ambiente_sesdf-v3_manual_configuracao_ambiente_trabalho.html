Manual de Configurações do seu Ambiente de Trabalho [ Voltar ]

Este manual tem por intenção permitir que o seu computador de trabalho esteja configurado corretamente para utilizar o Alphalinc em sua plenitude. Vamos explicar em poucos e simples passos como instalar/configurar ose seguintes requisitos de funcionamento do Sistema:

A) Navegador: Internet Explorer, versão 8.

Veja neste tópico, como certificar-se de que ele está configurado corretamente; e

B) Adobe Reader, versão 8 ou superior.

Como visualizar a versão instalada na sua máquina e atualizá-la, se necessário.

A) Internet Explorer 8

Este é o navegador homologado para a melhor utilização do Alphalinc. Siga os passos abaixo para garantir que está usando uma versão compatível do Internet Explorer.

1. Confirme a versão do seu Internet Explorer.

Dentro do navegador
 
 
O sistema abrirá a seguinte tela:

   
   
     
Para realizar uma consulta de materiais que estão vencidos ou perto de sua data de vencimento, siga os passos a seguir:
1º Passo: preencha os campos de filtro da tela de acordo com sua necessidade. Lembre-se que campos em amerelo são de preenchimento obrigatório.
  • Produto. Insira aqui um produto específico para a consulta. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os produtos cadastrados.
  • Grupo. Insira aqui um grupo de produtos específico para a consulta. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os grupos cadastrados.
  • Local. Insira aqui um local específico para a consulta. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os locais cadastrados.
  • Programa. Insira aqui um programa específico para a consulta. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os programas cadastrados.
  • Conta Contábil. Selecione aqui uma conta contábil para a consulta.
  • Período de... até. Selecione aqui um período de tempo específico para a consulta. Clique no botão [Data] para selecionar as datas iniciais e finais do período por meio de um calendário.
  • A vencer nos próximos. Insira aqui uma quantidade de dias.
  • e/ou que venceram nos últimos. Insira aqui uma quantidade de dias.
  • Formato. Selecione aqui entre as opções: "Contábil", "Físico" e "Contábil Detalhado".


Campos de Data
Campos de data possuem alguns atalhos importantes, são eles:
Data atual: digite o sinal . (ponto) e pressione a tecla "Enter" para que o sistema retorne a data atual;
Data do mês corrente: digite o dia do mês e pressione a tecla "Enter" para que o sistema retorne o mês e ano correntes;
Dias a contar da data atual: digite o sinal + (mais) ou - (menos) antes do número de dias em referência à data atual e pressione a tecla "Enter" para a data anterior ou posterior à data atual.
