Cadastro de Projetos e Programas [ Voltar]
Utilize este formulário para criar e editar cadastros de Projetos e Programas. Para acessar o formulário, vá ao menu "Cadastros" na página inicial e clique em "Projetos/Programas".
 

  
 

Ao clicar no link, o sistema abrirá a seguinte tela:


 

 

Selecione uma das duas opções abaixo para ver o passo-a-passo:

Cadastrar projeto ou programa
Editar projeto/programa cadastrado

 
 

Cadastrar projeto ou programa

1º Passo: preencha os dados do projeto/programa. 
  • Projeto. Não é preciso preencher este campo. Ele exibe por padrão um sinal de "+". Desta forma, assim que o cadastro é salvo, é-lhe atribuído automaticamente o próximo número de cadastro disponível;
  • Nome do Projeto. Informe aqui o nome do projeto ou programa que deseja cadastrar
  • Responsável. Informe aqui o nome da pessoa responsável pelo projeto/programa;
  • Início / Término. Se desejar, você pode informar aqui o período pelo o qual o projeto/programa será válido;
  • Observações. Deposite neste campo quaisquer informações adicionais sobre projeto/programa.
 
2° Passo: clique no botão  para salvar o cadastro.  
 
 


Editar projeto/programa cadastrado

1º Passo: para localizar o projeto que deseja editar, clique no botão da barra de botões.
 

 
 
 
Na tela de pesquisa do formulário "Cadastro de Projetos/Programas", são exibidos os filtros "Projeto" e "Nome do Projeto" para fácil localização do registro desejado. Configure os filtros da pesquisa para que o sistema retorne automaticamente todos os registros correspondentes. Para mais informações sobre outros recursos e funcionalidades da pesquisa, favor ver o manual Introdução ao Sistema.
 

 
  
  
Assim que o projeto/programa é selecionado, seu cadastro será aberto e exibidas as informações atuais.
 
 
2° Passo: edite as informações do cadastro.
  • Nome do Projeto. Informe aqui o nome do projeto ou programa que deseja cadastrar
  • Responsável. Informe aqui o nome da pessoa responsável pelo projeto/programa;
  • Início / Término. Se desejar, você pode informar aqui o período pelo o qual o projeto/programa será válido;
  • Observações. Deposite neste campo quaisquer informações adicionais sobre projeto/programa.
 
3° Passo: clique no botão  para salvar o cadastro.
