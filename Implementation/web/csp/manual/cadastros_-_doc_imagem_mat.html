Documentos e Imagens do Material [ Voltar ]


Esta tela permite que arquivos de texto, imagens e links externos sejam anexados a um produto.

Siga os passos abaixo para anexar um documento:

1º Passo: abra o cadastro de produtos. O formulário "Produtos" se encontra dentro do menu"Cadastros".


2° Passo: selecione o produto para o qual deseja carregar uma imagem ou documento. Digite no campo "Cód. Produto" o número identificador do produto ou clique no botão  [Procurar] para selecioná-lo a partir de uma lista. 

3º Passo: clique no botão para abrir o formulário "Documentos e imagens do Material". O botão  se encontra no canto superior direito do "Cadastro de Produtos".






Ao clicar no botão [Documentos e imagens do Material], a seguinte tela será exibida:





4º Passo: informe na tela os detalhes do documento ou imagem. Selecione o tipo de documento, adicione uma descrição para o documento ou imagem, selecione o arquivo que deseja carregar (upload) e, se necessário, especifique um link adicional da internet ou intranet (www.linkexterno.com/arquivo.pdf, por exemplo). Para selecionar o arquivo para o upload, veja abaixo:
  • Clique no botão  [Upload de arquivos] para visualizar a tela de upload;




  • Clique no botão para selecionar o arquivo que deseja carregar; e

  • Em seguida, após selecionar o arquivo desejado, clique no botão para realizar o upload.



 5º Passo: por último, clique no botão  [Salvar] para concluir a associação do arquivo ao produto.


Observação: se desejar carregar outros arquivos de imagem e/ou de texto, repita os passos 4 e 5 quantas vezes for necessário.
