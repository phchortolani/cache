Históricos de Compras  [ Voltar ]


Utilize este formulário para gerar um histórico de compras, que pode ser filtrado por produto, grupo, fornecedor e por um período.

O formulário "Históricos de compras" encontra-se dentro do menu "Relatórios Gerenciais".


Ao clicar no formulário, a seguinte tela será exibida: 


   

 
Para gerar o relatório, siga os passos abaixo: 

1º Passo: configure os filtros disponíveis de acordo com os dados que deseja obter. É necessário preencher ao menos um dos seguintes campos: "Produto", "Grupo" ou "Fornecedor".
 
2° Passo: clique no botão  para gerar o histórico.
 

 
 
3° Passo: para visualizar o relatório, clique no botão  na barra de botões. Para imprimi-lo, clique em seguida no botão  [Imprimir] do navegador.
