Estimativa de Preços [ Voltar ]

Utilize este formulário para definir produtos para os quais se deseja uma cotação de preços.

O formulário "Estimativa de Preços" encontra-se dentro do menu "Compras".


 

Ao clicar no nome do formulário, o sistema exibirá a seguinte tela:


 

Este manual contém os passos para exibe o passo-a-passo para as seguintes funcionalidades:

  1. Cadastrar ordem de fornecimento
  2. Cadastro e seleção de propostas
 
 

Criar estimativa de preços

1º Passo: informe os dados principais da estimativa. 

  • Estimativa. Este campo não precisa ser preenchido. Ele apresenta por padrão um sinal de "+", de forma que, quando o registro é salvo, é-lhe atribuído o próximo número de estimativa disponível;
  • Pedido de Compra. Se desejar gerar uma estimativa de preço para um pedido de compra específico, informe-o aqui. Clique no botão [Procurar] para selecioná-lo a partir de uma listagem contendo os pedidos de compra cadastrados no sistema;
  • Data. Informe aqui a data para quando é desejada a estimativa;
  • Observações. Deposite aqui informações adicionais sobre a estimativa.

2° Passo: clique no botão para salvar os dados informados. Caso um pedido de compra tenha sido informado, assim que a estimativa é salva, os produtos do pedido de compra em questão serão carregados na grade da parte inferior da tela. Para estimativas sem associação a um pedido de compra, o sistema exibirá na parte inferior da tela uma grade para a inclusão dos itens da estimativa.


 

Após informar os dados do produto, clique no botão [Novo] da grade (ver imagem abaixo) para especificar os demais produtos da estimativa.

 

  

Os outros botões disponíveis são:


Clique neste botão para excluir a linha selecionada.
Utilize este botão para realizar uma pesquisa entre todos os produtos cadastrados na estimativa.
Clique neste botão para copiar os dados da linha selecionada.
Este botão permite que o usuário exporte para uma planilha de Excel todos os dados de produtos contidos na estimativa.

 

 
4° Passo: clique no botão  para salvar a estimativa. 

5° Passo: clique na aba "Participantes" para cadastrar os participantes que comporão a estimativa de preços. Na parte inferior do formulário é exibida uma lista sugerida de fornecedores associados aos produtos da estimativa.


 

 

O sistema gera a lista de sugestões com as empresas que:

  • Fornecem o(s) produto(s), conforme o relacionamento definido no formulário "Cadastro de Produtos";
  • Já forneceram no passado o(s) produto(s), conforme o relacionamento feito no formulário "Ordem de Compra";
  • Possuem preço registrado no catálogo do banco de preços; e
  • Foram participantes em processos prévios de estimativa de preços.

É possível inserir manualmente os códigos dos participantes desejados na grade ou, de uma maneira mais simples, apenas clicar nas caixas de seleção dos fornecedores desejados (ver imagem abaixo). Caso você queira inserir TODOS os fornecedores sugeridos na lista de participantes, selecione a opção "Utilizar todos acima", disponível no final do formulário. 


  

 

6° Passo: clique no botão  para salvar a estimativa. Após cadastrados e salvos os participantes, você tem a possibilidade de convocá-los para a estimativa, tanto por fax quanto por email. Para solicitar a cotação por fax, clique no botão  [Imprimir] da barra de botões para visualizar a tela de impressão do pedido de cotação. Para imprimir a solicitação de cotação do fornecedor, clique no botão  [Imprimir] do navegador.

Para enviar a solicitação por email, clique no botão .  A seguinte tela será exibida:


 

  • Preencha as informações que devem constar no email. Os campos em amarelo são obrigatórios.
  • Quando todas as informações já tiverem sido preenchidas corretamente, clique no botão para enviar o email para todos os participantes.

Observação: O sistema só envia e-mail para os fornecedores que possuem um endereço de e-mail cadastrado no Cadastro de Fornecedores.

   

 
 

Cadastro e seleção de propostas

Após receber dos fornecedores a cotação dos produtos, é necessário cadastrá-las do sistema para poder gerar o mapa comparativo. 

1º Passo: para localizar a estimativa de preços que deseja consultar, clique no botão da barra de botões.
 


 
   
 
Na tela de pesquisa de registros de estimativas de preço, informe os dígitos iniciais do código da estimativa, nome, pedido de compra ou processo para que o sistema retorne automaticamente todas as transferências correspondentes. Para mais informações sobre outros recursos e funcionalidades da pesquisa, favor ver o manual Introdução ao Sistema.
 

 
 

Após localizar a estimativa desejada, selecione-a com um clique. O cadastro da mesma será aberto com todas as suas informações.

 

 

2° Passo: clique no botão para cadastrar as cotações de preços enviadas pelo fornecedor. Ao clicar no botão, a seguinte tela será exibida:
 

  

3º Passo: informe os dados da proposta. Na grade devem ser inseridas as propostas dos fornecedores para o produto selecionado. Após informar o código do fornecedor desejado, preço, marca e validade da proposta, clique em [Salvar]. Se existirem outros fornecedores com propostas para este mesmo produto, clique no botão [Novo] da grade para gerar uma nova linha e repita.

Observação: na parte inferior da tela são exibidos os participantes da estimativa. Você pode clicar em um deles para que seja adicionado automaticamente à grade de fornecedores. 
 

 
Depois de selecionar todos os fornecedores desejados, clique no botão [Salvar] e depois em [Voltar] para retornar à página principal. As propostas cadastradas serão mostradas na tabela da parte inferior da página:
 

 

Se for necessário cadastrar as propostas de outros produtos na estimativa de preços (como é no caso no exemplo acima), repita os passos acima.

5º Passo: clique no botão  para visualizar o resultado final da estimativa de preços. Para imprimir, clique no botão  [Imprimir] do navegador.

 

Observação: para a elaboração da estimativa de preços, é necessário que exista pelo menos uma proposta de preço para cada produto.
