Parâmetros de Reposição - Produto [ Voltar ]

Utilize esta tela para definir os parâmetros de reposição específicos a um produto. Serão definidas nesta tela variáveis de estoque ou tempo, fórmula de abastecimento, sazonalidade e demanda média.

O formulário "Parâmetros de Reposição - Produto" encontra-se dentro do menu "Planejamento". 


  

Ao clicar no formulário, o sistema exibirá a seguinte tela:


   
 

Os parâmetros aqui definidos serão utilizados para o cálculo de sugestão de requisições de compra para um produto específico em toda a rede. Estes parâmetros serão sobrepostos aos parâmetros de rede e locais existentes. 

 
1º Passo: configure na tela os parâmetros de reposição para o produto. Os campos em amarelo são obrigatórios.
  • Item. Informe aqui o código do produto para o qual deseja definir os parâmetros de reposição. Se necessário, clique no botão  [Procurar] para selecionar o produto a partir de uma listagem de produtos cadastrados;

  • ID. Este campo não precisa ser preenchido. Ele apresenta um sinal de "+" por padrão para que, desta forma, o sistema atribua a ele, automaticamente, o próximo número de registro disponível;

  • Descrição. Digite aqui a descrição para a definição dos parâmetros;

  • Ativo. Marque esta opção para ativar no sistema os parâmetros de reposição aqui definidos para o produto;

  • Parametrização. Marque uma das duas opções de parametrização de reposição: por Variáveis de Estoque  ou Variáveis de Tempo;

  • Variáveis de Estoque. Informe aqui o número de meses referentes a: Estoque de Segurança, Ponto de Ressuprimento e Estoque Máximo;

  • Variáveis de Tempo. Informe aqui o número de meses referentes a: Frequência de Abastecimento, Tempo de Compra ou Lead Time (Tempo de Ressuprimento);

  • Fórmulas de Abastecimento. A fórmula de abastecimento padrão é: Estoque Máximo - Estoque Virtual + Estoque de Segurança;

  • Cálculo. São exibidos nesta seção os valores calculados para Estoque de Segurança, Ponto de Ressuprimento, Estoque Máximo e Sugestão de Reposição;
  • Demanda Informada. Informe aqui a demanda média mensal para o produto;

  • Sazonalidade. Informe aqui dados específicos da sazonalidade da demanda para o produto. Exemplo, digite "20" no campo "Jan Fator (%)"  para indicar que deve haver um incremento de 20% nas sugestões para o mês de janeiro. Para decréscimos, inclua o sinal de negativo "-" antes do valor desejado;

  • Cálculo da Demanda Média. Informe neste campo o número de meses para o cálculo da demanda média no sistema. 

 
 
2° Passo: clique no botão  para salvar os parâmetros editados.
