Fazer Requisição [ Voltar ]

Para fazer uma requisição, clique em "Fazer Requisição" dentro do menu "Consumo".


 

Ao clicar no formulário, o sistema exibirá a seguinte tela:


 

Siga os passos abaixo para criar e confirmar uma requisição:

1º Passo: informe os dados iniciais para a requisição que deseja criar. Primeiro, especifique um local de origem no campo de mesmo nome. Se necessário, clique no botão  [Procurar] para selecioná-lo a partir de uma listagem contendo os locais cadastrados. No campo "Prioridade", informe o tipo de requisição: Emergencial, Periódica ou Eventual.

Observação: o campo "N°. da requisição" apresenta um sinal de mais "+" por padrão. Desta forma, assim que o registro é salvo, o sistema atribui à requisição o próximo número disponível. 

2º Passo: clique no botão  para salvar o documento. Assim que a requisição é salva, é exibida na parte inferior da tela uma grade para incluir os produtos requisitados.



3º Passo: especifique os produtos e respectivas quantidades requisitadas. Clique no botão [Procurar] para selecionar o produto a partir de uma listagem contendo todos os produtos cadastrados. Pressione a tecla "Enter" para ir de um campo a outro e para ir para a próxima linha. Você pode ainda clicar no botão  [Novo] da grade para criar novas linhas.

  
4º Passo: após especificar todos os produtos e quantidades, clique no botão  para salvar os dados inseridos.

5º Passo: se todos os dados inseridos estiverem corretos, clique no botão  para concluir a requisição. Se, ao contrário, desejar desfazer a confirmação, clique no botão  [Encerrar].
