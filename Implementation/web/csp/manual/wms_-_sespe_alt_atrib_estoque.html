Alteração de Atributos de Estoque [ Voltar ]

Utilize este formulário para editar algumas características de produtos atualmente em estoque. Para acessar o formulário, vá ao menu "WMS" na página inicial e clique em "Alteração de atributos do estoque " .


O sistema vai abrir a seguinte tela:


Para alterar os atributos de produtos existentes, siga os passos abaixo:

1º Passo: preencha os campos necessários:

  • Número: este campo apresenta por padrão um sinal de mais "+". Desta forma, assim que o registro é salvo, é-lhe atribuído o próximo número disponível.
  • Local: por padrão, neste campo o sistema automáticamente insere o código do local atual, mas este pode ser alterado. Caso queira saber mais a respeito de como mudar de local, consulte o manual Alterar Local.
  • Observações: deposite neste campo informações adicionais sobre a alteração. Este campo é de preenchimento voluntário.

2° Passo: clique no botão para salvar o registro. Uma grade para a inclusão dos produtos será exibida na parte inferior da tela. 


3º Passo: especifique os produtos cujos atributos devem ser alterados. Se necessário, clique no campo, o qual exibirá o botão [Procurar] (ver imagem abaixo), o qual permitirá que o produto seja selecionado a partir de uma listagem contendo todos os produtos cadastrados no sistema.


Após clicar no botão [Procurar], a seguinte janela de pesquisa será mostrada:


Para inserir mais produtos, simplesmente clique no botão [Novo] e repita o procedimento descrito no passo acima.

Depois que o código é inserido, uma tabela contendo os diversos lotes existentes do produto é mostrada, na parte inferior da tela:


4º Passo: selecione o lote adequado. É possível que existam no sistema vários lotes diferentes do produto selecionado. Clique com o cursor do mouse sobre o lote desejado, para que todos os dados sejam inseridos na tabela.


Observe como o sistema insere os dados da linha selecionada nos campos adequados da tabela.

5º Passo: faça as alterações desejadas. Os atributos que podem ser alterados são o lote, a data de vencimento, a marca e o programa.

6º Passo: clique no botão  [Salvar] para processar as alterações feitas nos produtos. Se desejar alterar os atributos de outros produtos, clique no botão  [Novo] e repita os passos acima.


Para retornar à tela principal, clique no botão .
