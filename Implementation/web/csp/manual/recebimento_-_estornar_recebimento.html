Estornar Recebimento [ Voltar ]

Este formulário permite que recebimentos de produtos realizados no sistema sejam desfeitos.

O formulário "Estornar recebimento" encontra-se dentro do menu "Recebimento".
 

Ao clicar no formulário, o sistema abrirá a seguinte tela:


Para estornar o recebimento de um determinado produto, siga os passos abaixo:
 
1° Passo: informe o número do recebimento desejado. Se necessário, clique no botão  [Procurar] para selecionar o recebimento a partir de uma lista contendo os recebimentos estornáveis.

Observação: apenas o recebimento mais recente de um determinado produto é passível de estorno e será exibido na tela de pesquisa de recebimentos. Logo, se o recebimento X de um produto é anterior ao recebimento Y para o mesmo produto, somente será possível selecionar e estornar o recebimento Y.


 

2º Passo: clique no botão  para salvar o estorno de recebimento. Após selecionar o recebimento que deseja estornar, é necessário salvar o registro para então inserir os produtos do estorno.

3º Passo: clique no botão  para selecionar os produtos que deseja estornar. Selecione com um clique o(s) produto(s) que deseja estornar.

Observação: em recebimentos efetuados com nota fiscal, somente será possível processar o estorno de todo o recebimento, isto é, não é permitido estornar apenas um dos produtos do recebimento.

4º Passo: clique no botão do navegador para retornar à tela principal do estorno.

5° Passo: confira os produtos selecionados para o estorno e clique no botão  para salvar o estorno. No caso de recebimento sem nota fiscal, se desejar adicionar outros produtos, repita o 3° passo. Para excluir algum produto já selecionado, clique na linha do mesmo e, em seguida, no botão  [Excluir] da grade.


Após qualquer alterações, será preciso salvar novamente o estorno.

4° Passo: após salvar os dados corretos do estorno, clique no botão  para concluí-lo.
