Pesquisar Requisições [ Voltar ]

Este formulário permite a pesquisa de requisições realizadas entre duas unidades de rede de saúde.

O formulário "Pesquisar Requisições" encontra-se dentro do menu "Transferencia".


 

Ao entrar no formulário, o sistema exibirá a seguinte tela:


  

 

Siga os passos abaixo para pesquisar uma Requisição:

1° Passo: utilize os favoritos para realizar a pesquisa de acordo com o perfil desejado.
   

   
     
     
  2° Passo: utilize os filtros da seleção de campos para facilitar a localização da Requisição desejada. A pesquisa de requisições exibe uma lista de todas as requisições cadastradas no sistema. Para mais informações sobre as funcionalidades da pesquisa, favor ver o manual Introdução ao Sistema.
 

   
 

  • Requisição. Digite aqui o código da Requisição, ou parte do código. Os resultados retornarão todos as requisições que contiverem o conteúdo digitado.
  • Local Abastecedor (Inicia com). Digite neste campo os caracteres iniciais da descrição do "Local Abastecedor". Os resultados retornarão todos as requisições que se iniciarem com o conteúdo digitado.
  • Local Abastecedor (Contém). Digite neste campo palavras-chave contidas na descrição do "Local Abastecedor". Os resultados retornarão todos os requisições que contiverem o conteúdo digitado ou clique no botão e selecione atraves da lista o local abastecedor desejado. 
  • Data (Igual a). Informe neste local a data desejada para pesquisa.
  • Status. Selecione aqui um status para a pesquisa de acordo com as opções disponiveis.
  • Programa. Selecione o programa desejado para presquisa.

Após utilizar os filtros, os resultados da pesquisa são exibidos automaticamente na tabela de resultados abaixo. Selecione com um clique a requisição desejada. Em seguida, A requisição será aberta na tela "Fazer Requisição" com a descrição completa da requisição realizada.
