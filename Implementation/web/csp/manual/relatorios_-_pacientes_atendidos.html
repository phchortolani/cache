Relatório de Pacientes Atendidos  [ Voltar ]


Utilize este formulário para gerar um histórico dos pacientes atendidos em um determinado local e período.

O formulário "Relatório de pacientes atendidos" encontra-se dentro do menu "Relatórios Gerenciais".


Ao clicar no formulário, a seguinte tela será exibida: 


    

 
Para gerar o relatório, siga os passos abaixo: 

1º Passo: configure os filtros disponíveis de acordo com os dados que deseja obter. Informe o local desejado e as datas que comporão o período da pesquisa.
 
2° Passo: clique no botão  para gerar o histórico.
 

 
 
 
3° Passo: para visualizar o relatório, clique no botão  na barra de botões. Para imprimi-lo, clique em seguida no botão  [Imprimir] do navegador.
