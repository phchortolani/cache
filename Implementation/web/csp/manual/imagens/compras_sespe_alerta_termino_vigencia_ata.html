Alerta de Término de Vigência de Ata [ Voltar ]

Utilize esta tela para consultar a vigência de atas. A pesquisa pode retornar atas vencidas, com vigência próxima ao fim ou ser restringida a um fornecedor, produto ou período.

O formulário "Alerta de Término de Ata" encontra-se dentro do menu "Compras".

  

 

Ao clicar no formulário, o sistema exibirá a seguinte tela:


   
  

Para gerar a listagem de atas e datas de vigência, siga os passos abaixo:

1º Passo: configure os filtros da pesquisa para localizar os contratos desejados. Os filtros disponíveis são:
  • Fornecedor. Se desejar gerar resultados para um fornecedor específico, informe aqui o seu código. Clique no botão  [Procurar] para selecionar o fornecedor a partir de uma listagem de fornecedores cadastrados;
  • Produto. Para gerar resultados para um produto específico, informe aqui o seu código. Clique no botão  [Procurar] para selecionar o produto a partir de uma listagem de produtos cadastrados;
  • Ata. Para localizar uma ataespecífica, informe aqui o seu código. Clique no botão  [Procurar] para selecionar a ataa partir de uma listagem de atas registradas;
  • Período de / até. Você pode selecionar nestes campos as datas que comporão o período da pesquisa.
Dicas para preenchimento dos campos de data:
    • Data atual: digite o sinal . (ponto) e pressione a tecla "Enter" para que o sistema retorne a data atual;
    • Data do mês corrente: digite o dia do mês e pressione a tecla "Enter" para que o sistema retorne o mês e ano correntes;
    • Dias a contar da data atual: digite o sinal + (mais) ou - (menos) antes do número de dias em referência à data atual e pressione a tecla "Enter" para a data anterior ou posterior à data atual. 
Exemplo: se a data atual é 20/10/2010, digite "+10" e pressione a tecla "Enter" para que o sistema retorne o dia 30/10/2010 
  • A vencer nos próximos ... dias. Período (em dias) para que o sistema retorne resultados de atas que vencerão neste período. Por exemplo, digite "90" para que a pesquisa consulte atas que vencerão nos próximos 90 dias. 
  • E/ou que venceram nos últimos ... dias. Período (em dias) para que o sistema retorne resultados de atas que venceram neste período. Por exemplo, digite "30" para que a pesquisa consulte atas vencidas nos últimos 30 dias. 
 
 
2° Passo: após configurar os filtros do alerta, clique no botão   para gerar os resultados.
 
 
 
 
Os seguintes símbolos são utilizados para facilitar a interpretação dos resultados:

  •  Contratos que vencerão nos próximos 30 dias;

  •  Contratos que vencerão entre 31 e 60 dias; e

  •  Contratos que vencerão entre 61 e 90 dias.
