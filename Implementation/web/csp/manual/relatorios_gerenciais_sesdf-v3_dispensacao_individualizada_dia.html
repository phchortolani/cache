Dispensação Individualizada por Dia  [ Voltar ] 

O formulário "Dispensação Individualizada por Dia" encontra-se dentro do menu "Movimentações - Relatórios Gerenciais".

 
 
Após clicar no formulário, o sistema abrirá a seguinte tela:

  
  
Siga os passos abaixo para gerar o relatório de dispensação individualizada por dia:

1º Passo: configure os detalhes do relatório. O campo "Data da dispensação" é obrigatório.
  • Centro de custo. Se desejar gerar o relatório para um centro de custo específico, informe-o aqui.  
  • Data da dispensação. Informe aqui o dia para o qual deseja visualizar o relatório.
  • Ordenar por grupo. Marque esta opção se desejar que os resultados sejam ordenados pelo grupo de materiais.
  • Ordenar produto por Código/Descrição. Selecione uma das duas formas de ordenamento possíveis: pelo código ou pela descrição do material.
  • Exibir pacientes. Marque esta opção se desejar que o nome do paciente referente a cada dispensação seja também exibido no relatório.
    
2º Passo: clique no botão  processar o relatório.
