Meus Contratos [ Voltar ]

 Utilize este formulário para visualizar e ter acesso a todos os contratos em que o usuário é executor.
 
O formulário "Meus contratos" encontra-se no menu "Contratos".




 

Após clicar no formulário, o sistema abrirá a seguinte tela:


 
 
  
A tela exibe os dados principais de todos os contratos em que o usuário é executor.
 
  • Para acessar um determinado contrato, clique no número do contrato desejado ou em um dos campos a ele relacionado.
 

 
 
  •  Para rápido acesso à tela "Registro de notificação de executores", apenas clique no campo "Inserir notificação" correspondente ao contrato para o qual deseja criar a notificação.
 
 
Para mais informações sobre como registrar uma notificação, favor ver manual "Notificação de executores".
