Estornar Recebimento  [ Voltar ]

Esta tela permite o usuário estornar recebimentos já processados.

Para acessá-lo, vá ao menu "Recebimento - Recebimento" da tela inicial e clique em "Estornar Recebimento". 
   
   
   
   Ao clicar no menu, a seguinte tela será exibida:

 
 
 
Execute os passos abaixo para gerar a guia de remessa:

1º Passo: preencha o formulário de estorno. Lembre-se que todos os campos amarelos .
  • No. Estorno. Este campo por padrão vem com o sinal "+". Isto indica que quando salvo o formulário é-lhe atribuído o próximo número disponível de estorno.
  • No. Recebimento. Indique aqui o código do recebimento que será estornado. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os recebimentos cadastrados.
  • Observações. Adicione aqui o motivo da necessidade de estornar o recebimento.

Após confirmar que todos os campos estão corretamente preenchidos, clique no botão [Salvar]. O botão [Inserir Produtos] estará disponível.

 
    
 
2° Passo: clique no botão [Inserir Produtos]. Será aberta uma tela de pesquisa de produtos do recebimento em questão. Conforme imagem abaixo.

 
 
 
3º Passo: selecione os produtos que serão estornados na tela de pesquisa. Utilize os filtros da pesquisa para facilitar a localização do produto que será estornado. Depois de localizado selecione-o com um clique. Lembre-se que podem ser estornados mais de um produto ao mesmo tempo. Após selecionar os produtos clique no botão [Salvar] para armazenar as informações, o botão [Processar] estará disponível depois deste passo.
 
 
 

4º Passo: clique no botão [Processar] para processar o estorno. Caso queira imprimir o relatório do estorno clique no botão [Relatório].
