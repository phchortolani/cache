Consultar Recebimentos  [ Voltar ]

Utilize este formulário para realizar recebimentos e incorporação de produção para o Almoxarifado Central.

Para acessá-lo, vá ao menu "Recebimento - Recebimento" da tela inicial e clique em "Recebimento / Incorporação-Produção". 
   
   
   
Ao clicar no menu, a seguinte tela será exibida:

 
 
 
Execute os passos abaixo para realizar o recebimento:

1º Passo: preecha os campos do formulário. Lembre-se que campos amarelos são obrigatórios.
  • No. Entrada. Por padrão, este campo contém o sinal "+". Desta forma, quando o registro é salvo, é-lhe atribuído o próximo número disponível à entrada.
  • Fornecedor. Selecione aqui o fornecedor da entrada. Tenha em mente que somente fornecedores do Almoxarifado Central irão aparecer aqui.
  • Data de Recebimento. Indique aqui a data de recebimento da entrada. Clique no botão [Data] para selecioná-la a partir de um calendário.
  • Mat. Recebedor. Adicione aqui a matrícula do responsável pelo recebimento.
  • No. Doc. Origem. Adicione aqui o número do documento que originou o recebimento da entrada.
  • Valor Total do Doc. Indique aqui o valor total do recebimento em questão.
  • Observações. Adicione aqui quaisquer informações adicionais sobre a entrada.

Campos de Data
Campos de data possuem alguns atalhos importantes, são eles:
Data atual: digite o sinal . (ponto) e pressione a tecla "Enter" para que o sistema retorne a data atual;
Data do mês corrente: digite o dia do mês e pressione a tecla "Enter" para que o sistema retorne o mês e ano correntes;
Dias a contar da data atual: digite o sinal + (mais) ou - (menos) antes do número de dias em referência à data atual e pressione a tecla "Enter" para a data anterior ou posterior à data atual. 
 
 
 
Caso todos os campos estejam preenchidos corretamente, clique no botão [Salvar] para armazenar as informações.

2° Passo: preencha a grade de produtos com os dados dos produtos da entrada. Assim que é salva as informações, aparece uma grade para a inserção dos produtos relacionado ao recebimento. Conforme figura abaixo.

  
 
 
Preencha os dados dos produtos em questão.
  • Código / Descrição. Entre aqui com o código do produto a ser inserido. Clique no botão [Pesquisar] para selecioná-lo por meio de uma listagem contendo todos os produtos cadastrados e permitidos para o Almoxarifado Central.
  • Quantidade. Informe aqui a quantidade de itens do produto em questão.
  • Preço Unitário. Adicione aqui o valor do produto em questão.
  • Grupo. Selecione aqui o grupo do produto.
Após se certificar que inseriu todos os produtos corretamente, clique no botão [Salvar] para armazenar as informações sobre os produtos.
