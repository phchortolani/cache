Source Control Dependencies

In order to load and compile all source control related artifacts, the system must honor some existing dependencies. Unfortunately, Alphalinc presents some circular dependencies. In order to cope with that, some Cache classes generated by @NM are also kept under source control, because the system needs them to generate the first @NM classes!
