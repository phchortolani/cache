Farmárcia de Média Complexidade

Em algumas situações o médico pode prescrever um tratamento continuado, onde o paciente vai precisar periodicamente retirar medicamentos de uma farmácia especializada. O sistema vai controlar o consumo e a dispensação desses medicamentos.

Este caso de uso é composto pelos seguintes casos de uso

  1. Cadastramento de tratamento continuado - Média Complexidade
  2. Cadastramento de entrega de medicamentos para tratamento continuado - Farmácia Especializada ao paciente - Média Complexidade
