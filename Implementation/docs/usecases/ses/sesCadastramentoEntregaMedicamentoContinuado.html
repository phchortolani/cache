Cadastramento de entrega de medicamento para tratamento continuado

Utilizado por: Farmácia de Média Complexidade

Atendende de farmácia especializade de média complexidade cadastra dispensações para tratamento continuado

Usuários

Atendente de farmácia especializada

Pré-requisitos

Usuário logado

Passos

  1. O usuário seleciona opção de dispensação para tratamento continuado
  2. O usuário seleciona o paciente
  3. O sistema informa prescrição, quantidade de medicamento usado e quantidade restante
  4. O usuário seleciona quantidade para dispensação
  5. O usuário confirma
  6. O sistema cadastra a dispensação, diminuindo o estoque
