Cadastramento de tratamento continuado

Utilizado por: Farmácia de Média Complexidade

Quando o tratamento inclui medicamentos de média coplexidade, o médico faz uma prescrição mas também cadastra o tratamento continuado no sistema

Usuários

Médicos cadastrados

Pré-requisitos

Usuário logado

Passos

  1. O usuário seleciona opção de cadastro de tratamento continuado
  2. O sistema solicita diagnóstico (CID e descrição)
  3. Usuário informa CID e descrição breve
  4. O sistema solicita prescrição (medicamentos, dosagens, duração)
  5. O usuário informa lista de medicamentos,dosagens e duração
  6. O usuário confirma
  7. O sistema cadastra o tratamento continuado
