Alphalinc Software Development Environment - Documentation

This is the starting point for Alphalinc documentation. This documentation is versioned and can be found in the directory docs within the implementation part of the project. Update the contents using bzr update and then point the browser to yourProjectDir/Implementation/docs/index.html.

Please choose the appropriate set of documents.

  1. Overview - how the pieces fit together.
  2. ASDE vs Core vs Customization - understand how ASDE differentiates core and customizations.
  3. Installation and configuration - how to get started.
  4. Namespace migration - how to migrate an existing namespace to ASDE.
  5. Using the tools - how to do everyday tasks.
  6. Script release framework - how to create a script to install a new version.
  7. Troubleshooting guide - common problems and solutions.

For making changes to the Alphalinc Software Development Environment, improvements, bugfixes or undertanding it in depth, the following docs are available:

  1. System maintenance documentation - documents related to source control and development environment documentation. The focus is on system maintenance.
  2. To-do list - things that are still missing from the Alphalinc Development Environment (please edit it with wordpad or any simple html editor).
  3. Solved issues and to-do list items - historical tracking of the items that were solved.
  4. Issues and considerations - general considerations, issues and concerns about the Alphalinc Development Environment.
  5. History of implementation decisions - historical tracking of the decisions that were taken regarding the ASDE development.
